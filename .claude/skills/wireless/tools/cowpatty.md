# coWPAtty

- **Category**: Wireless / WPA Cracking
- **Risk Level**: 🟡 Medium

---

## Description

A command-line WPA/WPA2 PSK offline dictionary cracker. Cracks captured 4-way EAPOL handshakes against a password wordlist. Also supports precomputed PMK hash tables (generated by `genpmk`) for faster cracking of specific SSIDs. CPU-based; use when GPU is unavailable, or as a complement to aircrack-ng for cross-validation.

## Installation

```bash
sudo apt install cowpatty
```

## Parameter Reference

| Parameter | Description |
|-----------|-------------|
| `-f <wordlist>` | Dictionary file |
| `-r <pcap>` | Packet capture file |
| `-s <ssid>` | Network SSID (enclose in quotes if SSID includes spaces) |
| `-d <hashfile>` | Hash file (genpmk) |
| `-v` | Print verbose information (more -v for more verbosity) |
| `-c` | Check for valid 4-way handshake frames only (no cracking) |

## Common Commands

```bash
# Crack WPA handshake with rockyou.txt
cowpatty -f /usr/share/wordlists/rockyou.txt -r capture.cap -s "TargetSSID"

# Verbose mode (show each attempt)
cowpatty -f wordlist.txt -r capture.cap -s "TargetSSID" -v

# Generate precomputed PMK hash file for a specific SSID
genpmk -f /usr/share/wordlists/rockyou.txt -d pmkhash.db -s "TargetSSID"

# Crack using precomputed PMK file (faster for repeated attacks on same SSID)
cowpatty -d pmkhash.db -r capture.cap -s "TargetSSID"
```

## Notes & Tips

1. The capture file must contain a complete 4-way EAPOL handshake — verify with aircrack-ng first.
2. cowpatty is significantly slower than hashcat GPU cracking; prefer `hashcat -m 22000` when a GPU is available.
3. The precomputed PMK approach (`genpmk` + `cowpatty -d`) is only worthwhile when cracking the same SSID across multiple wordlists.
4. Recommended workflow: capture with airodump-ng → convert with hcxpcapngtool → crack with hashcat -m 22000.
5. Use cowpatty for CPU-only environments or as a fallback when hashcat is unavailable.
